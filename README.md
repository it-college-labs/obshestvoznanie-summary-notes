# Нейроархив обществознания

Интерактивный архив конспектов по обществознанию: стартовая сцена с ботом,
canvas-раскрытие проводника недель, spring-анимации папок и красивые MDX-статьи.

## Команды

```bash
npm install
npm run dev
npm run lint
npm run build
```

Контент добавляется через `src/content/articles.ts` и MDX-файлы в
`src/content/notes`. Правила для следующих агентов лежат в
`docs/agent-content-guide.md`.
